# The Intensive 6-Day Protocol Repository

This repository contains the documentation and theoretical framework for the "Big Three" herbal protocol, as popularized by researchers like Dr. Hulda Clark.

## Information Overview
This project archives the theory that cancer can be viewed as an inflammatory response to parasitic infestation and outlines the historical 6-day herbal "shock" treatment.

## Repository Contents
* **index.html**: The main webpage featuring the 6-day dosage table and theory.
* **The Triple Threat**: Detailed breakdown of Wormwood, Cloves, and Black Walnut.

## How to Deploy Your Own Version
1. Create a new repository on GitHub.
2. Upload `index.html` and this `README.md`.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, set the source to "Deploy from a branch" and select `main`.
5. Your site will be live at: `https://[your-username].github.io/[repo-name]/`
